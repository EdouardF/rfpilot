# RFPilot

AI Government RFP Responder

## Features
- RFP management with deadline tracking
- AI-powered proposal drafting
- Compliance checklist with pass/fail/warning
- Priority-based RFP triage

## Tech Stack
- React + Vite + TypeScript + TailwindCSS v4 + Zustand + Vitest

## Getting Started
```bash
npm install && npm run dev
```

## Testing
```bash
npx vitest run
```
